## 1.2. 粗体、斜体、删除线

| 样式   | 语法       | 例子               | 效果             |
| ------ | ---------- | ------------------ | ---------------- |
| 1      | 2          | 3                  | 4                |
| 粗体   | `**`or`__` | `**粗体实例**`     | **粗体实例**     |
| 斜体   | `*`or`_`   | `*斜体实例*`       | *斜体实例*       |
| 删除线 | `~~`       | `~~删除线实例~~`   | ~~删除线实例~~   |
| 粗斜体 | `***`      | `***粗斜体实例***` | ***粗斜体实例*** |

## 1.3. 引用

    >我是一段引用
